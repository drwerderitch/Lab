Settings & About

How it works

A dot travels every string in the pattern exactly once, playing a xylophone tone at each nail. Shorter strings = higher pitch. When all strings are visited, the sequence loops.

Traversal method

The path is pre-computed before playback using Fleury's algorithm — a greedy walk that always prefers unvisited segments and avoids bridges. This guarantees every string gets played before the loop restarts.

Combinatorics

With 12 nails, there are C(12,2) = 66 possible segments.

2⁶⁶ ≈ 73 quintillion

Distinct patterns possible — more than any student could ever exhaust.

Using Burnside's lemma with the dihedral group for the current nail count, visually unique patterns (accounting for rotation and reflection) are roughly 1/(2N)th of that — still astronomical.

On the horizon

  • Syncopation and flanger — modulate dot speed or pitch across the traversal to create rhythmic and timbral variation
  • MIDI output — trigger external synths via Web MIDI API, mapping each nail hit to a note-on event
  • Web Audio API routing — expose the oscillator output so it can feed into browser-based synths like Tone.js

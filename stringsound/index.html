How it works

A dot travels every string in the pattern exactly once, playing a xylophone-style tone at each nail. Shorter strings = higher pitch by default. When all strings are visited, the sequence loops.

Traversal method

The path is pre-computed using Fleury's algorithm — a greedy walk that prefers unvisited segments and avoids bridges — guaranteeing every string gets played before the loop restarts.

Combinatorics

With 12 nails, there are C(12,2) = 66 possible segments.

2⁶⁶ ≈ 73 quintillion

Distinct patterns possible before any repeat.

Using Burnside's lemma with the dihedral group for the current nail count, visually unique patterns are roughly 1/(2N)th of that — still astronomical.
